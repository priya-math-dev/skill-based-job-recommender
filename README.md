Skill-Based Job Recommendation System

📌 Project Description

This is a Python project that recommends suitable IT job roles based on the user's skills.
The program compares user skills with the required skills for different job roles and calculates the skill match percentage.
It also visualizes the result using a bar chart with the help of Matplotlib.

🎯 Features

- Takes user skills as input
- Compares skills with different IT job roles
- Calculates skill match percentage
- Suggests the best matching job role
- Displays a bar chart visualization for skill match

🛠 Technologies Used

- Python
- Matplotlib
- Basic Data Structures (Dictionary, List)

📊 How It Works

1. The user enters their skills.
2. The program stores job roles and required skills in a dictionary.
3. It compares user skills with each job role.
4. Calculates the match percentage.
5. Displays the best matching job role.
6. Shows a bar chart of job roles vs skill match percentage.

▶️ How to Run the Project

1. Install Python.
2. Install Matplotlib:
   pip install matplotlib
3. Run the Python file in PyCharm.
4. Enter your skills when prompted.

📷 Output

- Displays recommended job role
- Shows skill match percentage
- Generates a bar chart visualization
